## 프로젝트 소개
Flask와 Jinja2를 사용해서
파이썬 데이터를 HTMl 화면에 출력하는 연습을 하는 프로젝트입니다.

## 지난 시간에 배운 내용
- Flask 기본 실행
- templates 폴더 사용
- index.html 만들기
- render_template()로 HTML 연결하기
- 파이썬 딕셔너리 데이터를 HTML에 출력하기
- 리스트 데이터 만들기
- for 문으로 화면 자동 생성하기
- if 문으로 화면 다르게 출력하기
